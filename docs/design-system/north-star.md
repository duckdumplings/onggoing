> 작성: 2026-05-29.
> 목적: 옹고잉 스마트 물류 플랫폼의 디자인 시스템이 향해야 할 방향을 1페이지로 못 박는다.
> 신규 컴포넌트/시각 시안은 §0 정체성 5축 컴플라이언스 체크리스트로 판정한다.
>
> **Living Document**: §0 원칙은 시점-무관, §3 현재 상태 스냅샷은 Phase 전환마다 갱신.

## 0. 제품 정체성 (5축)

> 정체성 없이 "조화/절제"라는 추상으로 만들면 익숙한 슬롭으로 회귀한다.
> 본 §0에서 못 박는다.

### 5축 정의

| 축 | 결과 | 의미 |
|---|---|---|
| 1. **톤** | 진중 | B2B 어드민. 명사체 우선("경로를 최적화할까요?" OK / "확인하시겠습니까?" X). 토스페이먼츠·채널톡 톤. |
| 2. **표현** | 정밀 | 숫자/표/지도가 주인공. `.tabular` (tabular-nums + slashed-zero) 강제, 단위(원/km/분) 정렬·축소 표시. 자비스·Stripe Sigma 톤. |
| 3. **시대감** | 모던 | 둥근 모서리(`--radius: 12px`) · 여백 · 세리프 절제. 토스·카카오모빌리티 어드민 시대감. |
| 4. **장식** | 표현 베이스 + 미니멀 조화 | 글래스 tier(launcher/card/canvas) 도입, 단 모든 카드가 깊은 게 아니라 **강조점에서만 깊이**. |
| 5. **에너지** | 활기 | 정적 X. 마이크로 인터랙션, 필터 transition, optimistic feedback. 단 §1(진중)과 충돌하지 않게 timing curve는 절제. |

### 한 문장 정의

> **"정밀하지만 살아있는 B2B 모던 어드민 — 지도/표/숫자가 본문"**
>
> = 토스페이먼츠의 진중함 + Stripe Sigma의 정밀 + 카카오모빌리티 어드민의 도메인 친화

### 정체성 5축이 디자인 결정에 작동하는 방식

| 영역 | 결정 | 근거 axis |
|---|---|---|
| 베이스 | 라이트 글래스 (`.glass-card` 기본 tier) | 톤(진중) + 시대감(모던) |
| 타이포 | 숫자가 주인공. `.tabular` 강제, `text-3xl font-semibold` 큰 단위 표시 | 표현(정밀) |
| 머티리얼 | 글래스 tier 시스템 도입(launcher/card/canvas), 베이스는 평면 + 1px border, 강조점에서만 깊이(canvas) | 장식(표현+미니멀 조화) |
| 모션 | focus 링 · transition 색상 · optimistic feedback. timing curve 절제 (200~250ms ease-out) | 에너지(활기) + 톤(진중) |
| 컬러 | 토큰 시맨틱 + 강조점에서만 `chart-1/2` 사용. 모든 카드 accent bar/grad shadow 금지 | 톤(진중) |

### 신규 컴포넌트 컴플라이언스 체크리스트

5개 모두 통과해야 정체성 일치 판정:

- [ ] **진중**: 카피 명사체 / 모션 timing 절제 / 강조 색 자제
- [ ] **정밀**: 숫자에 `.tabular` / 큰 숫자 디스플레이 weight / 단위 정렬·축소
- [ ] **모던**: 둥근 모서리(`rounded-lg/xl`) + 여백 / 본문 세리프 금지
- [ ] **표현+조화**: 강조점에만 깊이감, 베이스 카드는 평면 + 1px border
- [ ] **활기**: 화면당 마이크로 인터랙션 1개 이상 (focus 링 / transition / optimistic feedback)

## 1. 자산 vs 슬롭 — 판정 기준

> **판정 기준**: "이 효과를 빼면 어떤 정보가 사라지는가?"
>
> - 답할 수 있다 → 자산
> - 답할 수 없다 → 슬롭

### 자산 (허용)

- **카드 스택 perspective** — "미처리 작업이 누적되어 있다"는 메타포
- **상태 dot soft glow** — 운행중/대기/완료 즉시 인지
- **글래스 tier 위계** — launcher(주변) → card(본문) → canvas(강조) 의 시각 계층
- **focus 링** — 키보드 네비게이션 가시화
- **layoutId 활성 전환** — 선택 상태 연속성
- **지도 경로 폴리라인 그라디언트** — 출발→도착 방향 정보 전달
- **숫자 디스플레이 weight 변별** — KPI 큰 숫자 + 단위 축소 + 트렌드 작게

### 슬롭 (금지)

- **모든 카드 좌측 accent bar** — 위계 없음, 인플레이션
- **모든 항목 컬러 shadow** — 깊이 인플레이션
- **ambient blob 4개 + noise.svg overlay** — 다크 의존 hero 슬롭
- **다크 hero 강제** — 본문은 라이트 글래스가 가독성 우월
- **hero gradient 박스 (`bg-gradient-to-br from-blue-500 to-blue-600`)** — 의미 없는 브랜드 장식
- **UI/코드/주석 이모지** — 운영 도구 시야 갈라짐
- **화살표 항상 ↑ 트렌드** — 음수 trend여도 ↑ 표시는 거짓 시그널
- **인라인 글래스 (`bg-white/55 backdrop-blur-xl`)** — tier 시스템 우회

## 2. 한국어 카피 톤

토스/채널톡/카카오모빌리티 어드민 톤.

| 슬롭 | 대안 |
|---|---|
| "확인하시겠습니까?" | "경로를 최적화할까요?" / "견적서를 생성할까요?" |
| "오류가 발생했습니다" | "Tmap 호출에 실패했어요" / "견적 계산이 중단됐어요" |
| "잠시만 기다려 주세요" | "경로 최적화 중…" / "견적 계산 중…" |
| "환영합니다!" | (제거) — 첫 화면은 행동(주소 입력/파일 업로드) 제시 |
| "데이터가 없습니다" | "최근 배차 이력이 없어요" / "저장된 견적이 비어 있어요" |
| "성공적으로 처리되었습니다" | "견적서가 저장됐어요" / "경로 최적화가 끝났어요" |

원칙:
- 친절체("~요"), 명령형 지양
- 도메인 명사 명시 (배차/견적/경로/배송원/경유지/Tmap)
- 다음 행동 제시 (오류 카피에 "다시 시도", "고객센터")

## 4. 레퍼런스 정책

### 채택 (B2B 배차/물류 운영 도메인 일치)

- **카카오모빌리티 어드민** — 라인/권역 관리, 지도 운영 UI
- **티맵 비즈니스 어드민** — 다중 경유지, ETA 표시
- **토스페이먼츠 어드민** — 견적·요금제 표 가독성
- **자비스(Jobis)** — 정산/회계 표
- **채널톡 어드민** — B2B 정보 밀도

### 제외 (의도적)

- Linear / Vercel / Stripe 대시보드 — 글로벌 dev tool 미감, 배차 도메인과 어긋남
- shadcn examples / 21st.dev 다크 hero 클론 — AI 슬롭 직접 출처
- Apple Maps / Google Maps 일반 사용자 UI — 운영자용 다중 경유지와 정보 밀도 다름

## 5. 시각 변화 등급 (변경 사이즈 가이드)

> 변경 작업을 단독 PR로 묶을지, 묶음 PR로 갈지 판단 기준.

| 등급 | 기준 | 진행 방식 |
|---|---|---|
| **polish** | <12% — 폰트 weight, 1px stroke, 색 미세 shift, 토큰 치환 | 단독 PR 금지. backlog 누적 후 분기 1회 정리 PR |
| **noticeable** | 12~30% — 폰트 크기 단계, 색 채도 30%, 간격 8→12px, micro-interaction 추가 | 단독 PR · 시각 검증 사이클 |
| **obvious** | 30%+ 또는 구조 변화 — master-detail, 새 컴포넌트, 레이아웃 재구성 | **사전 컨펌 필수** · 단계별 진행 |

자체 평가 체크리스트 (3개 이상 "예" → noticeable 이상):

- [ ] 변경된 RGB가 *눈으로 보이는* 만큼 다른가? (토큰 치환 RGB 동일 → polish)
- [ ] 변경 영역이 첫 진입 시점에 보이는가? (hover/click 의존이면 polish)
- [ ] viewport 점유 비율 변화가 ≥5%인가?
- [ ] 정보 구조나 인터랙션 흐름이 변하는가?
- [ ] 모르는 사용자가 "어딘가 달라졌다"고 즉시 인지하는가?
